# modular-storage-box
3D Printable Modular Storage Box

# WARNING - WORK IN PROGRESS

## Print Settings

### Boxes
#### 1x1 or 1x2 box:
Use Vase Mode or Single shell
.2 Layer height
3 layers on bottom
PLA
#### 2x2 or 1x4 or larger
Use 2 shells
3 layers on bottom

### Grid
Normal Settings should suffice
